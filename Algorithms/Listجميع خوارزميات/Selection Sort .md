# Selection Sort

## 🔵 ما هي خوارزمية Selection Sort؟

خوارزمية **الترتيب بالاختيار (Selection Sort)** هي طريقة بسيطة لترتيب العناصر.

في كل دورة، تبحث عن **أصغر عنصر** في الجزء غير المرتب من القائمة، وتقوم بتبديله مع أول عنصر في هذا الجزء.

---

## ✅ خطوات تصميم الخوارزمية:

| رقم | الخطوة |
| --- | --- |
| 1 | تعريف دالة باسم `selectionSort`. |
| 2 | تمرير القائمة المطلوب ترتيبها كوسيط. |
| 3 | استخدام حلقة تمرّ على كل عنصر في القائمة (عدا الأخير). |
| 4 | في كل دورة، نفترض أن العنصر الحالي هو الأصغر. |
| 5 | نبحث في بقية العناصر عن عنصر أصغر. |
| 6 | إذا وجدنا عنصرًا أصغر، نحفظ فهرسه. |
| 7 | بعد انتهاء البحث، إذا تغيّر الفهرس، نبدّل العنصر الأصغر مع العنصر الحالي. |
| 8 | تتكرر العملية حتى تصبح القائمة مرتبة. |

---

## 💻 الكود الكامل مع الشرح داخل التعليقات:

```dart

// الخطوة 1: تعريف دالة باسم selectionSort لترتيب القائمة
void selectionSort(List<int> list) {
  int n = list.length;

  // الخطوة 3: نبدأ حلقة خارجية تمر على كل عنصر (عدا الأخير)
  for (int i = 0; i < n - 1; i++) {
    // الخطوة 4: نفترض أن العنصر الحالي هو الأصغر
    int minIndex = i;

    print("🔍 الجولة ${i + 1}: نبحث عن أصغر عنصر من الفهرس $i إلى ${n - 1}");

    // الخطوة 5: نبدأ حلقة داخلية للبحث عن العنصر الأصغر
    for (int j = i + 1; j < n; j++) {
      print("مقارنة بين ${list[minIndex]} و ${list[j]}");

      // الخطوة 6: إذا وجدنا عنصرًا أصغر، نحدث مؤشر الأصغر
      if (list[j] < list[minIndex]) {
        minIndex = j;
        print("→ وجدنا عنصر أصغر: ${list[minIndex]} عند الفهرس $minIndex");
      }
    }

    // الخطوة 7: إذا تغير مؤشر الأصغر، نقوم بالتبديل
    if (minIndex != i) {
      print("↔️ تبديل ${list[i]} مع ${list[minIndex]}");

      int temp = list[i];
      list[i] = list[minIndex];
      list[minIndex] = temp;

      print("📌 الحالة بعد التبديل: $list");
    } else {
      print("✅ لا حاجة للتبديل. العنصر في مكانه الصحيح.");
    }

    print("📦 نهاية الجولة ${i + 1}: $list\n");
  }
}

// دالة main لتجربة الخوارزمية
void main() {
  // قائمة غير مرتبة
  List<int> numbers = [29, 10, 14, 37, 13];

  print("🔢 القائمة قبل الترتيب:");
  print(numbers);

  // استدعاء دالة الترتيب
  selectionSort(numbers);

  print("✅ القائمة بعد الترتيب:");
  print(numbers);
}

```

## 🎯 مثال عملي (خطوة بخطوة):

لو عندك القائمة: `[29, 10, 14, 37, 13]`

- الجولة 1:
    - أصغر عنصر هو `10` → تبديل مع `29` ⇒ `[10, 29, 14, 37, 13]`
- الجولة 2:
    - أصغر من `29` هو `13` → تبديل ⇒ `[10, 13, 14, 37, 29]`
- الجولة 3:
    - أصغر من `14` هو نفسه ⇒ لا تبديل
- الجولة 4:
    - أصغر من `37` هو `29` → تبديل ⇒ `[10, 13, 14, 29, 37]`
- النتيجة النهائية ✅

---

## 🧠 تحليل الأداء:

| الحالة | الزمن المستغرق (Time Complexity) |
| --- | --- |
| أفضل حالة | O(n²) |
| أسوأ حالة | O(n²) |
| الفائدة | عدد عمليات التبديل أقل من Bubble Sort |
