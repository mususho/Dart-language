```dart
1. خوارزميات الشبكات الأساسية (Graph / Network Algorithms)
بحث واستكشاف الشبكة:

Breadth-First Search (BFS) – البحث بالعرض.

Depth-First Search (DFS) – البحث بالعمق.

أقصر مسار (Shortest Path):

Dijkstra’s Algorithm – أقصر مسار من نقطة واحدة لكل النقاط.

Bellman-Ford Algorithm – أقصر مسار مع إمكانية الأوزان السالبة.

Floyd-Warshall Algorithm – أقصر مسار بين كل الأزواج.

A Algorithm* – مسار أمثل مع استخدام تقدير heuristics.

أشجار الامتداد (Spanning Trees):

Kruskal’s Algorithm – شجرة امتداد أقل تكلفة.

Prim’s Algorithm – شجرة امتداد أقل تكلفة.

تدفقات الشبكة (Network Flow):

Ford-Fulkerson Algorithm – أقصى تدفق في الشبكة.

Edmonds-Karp Algorithm – نسخة محسنة من Ford-Fulkerson.

Dinic’s Algorithm – أقصى تدفق مع كفاءة أعلى للشبكات الكبيرة.

الاتصال والمكونات (Connectivity):

Tarjan’s Algorithm – لإيجاد الجسور والنقاط الحرجة.

Kosaraju’s Algorithm – لإيجاد المكونات القوية المتصلة.

Union-Find / Disjoint Set – لإدارة المجموعات المتصلة.

2. خوارزميات التوجيه في الشبكات (Routing)
RIP (Routing Information Protocol) – خوارزمية تعتمد على مسافات القفز.

OSPF (Open Shortest Path First) – خوارزمية تعتمد على خوارزمية Dijkstra.

BGP (Border Gateway Protocol) – يعتمد على المسار وسياسات الشبكة.

Link-State Routing Algorithms – تبني خريطة كاملة للشبكة.

Distance-Vector Routing Algorithms – تعتمد على تحديث الجيران فقط.

3. خوارزميات تحسين الشبكات / التحميل (Network Optimization)
Maximum Flow / Minimum Cut

Minimum-Cost Flow

Edmonds’ Algorithm – لتعظيم التطابق (Matching) في الرسوم البيانية.

Hungarian Algorithm – مطابقة مثالية في الرسوم الثنائية.

4. خوارزميات خاصة بالشبكات اللاسلكية / P2P
Ad hoc On-Demand Distance Vector (AODV) – توجيه للشبكات اللاسلكية.

Dynamic Source Routing (DSR) – توجيه ديناميكي للشبكات اللاسلكية.

Chord / Kademlia – خوارزميات شبكات موزعة P2P.
```
